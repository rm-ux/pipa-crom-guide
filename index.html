Guide: Unlocking Bootloader, Flashing Recovery & ROM on Xiaomi Pad 6

Prerequisites

  • Xiaomi Pad 6 (HyperOS)
  • ADB and Fastboot installed on PC (Download here)
  • Mi Unlock Tool (Download here)
  • USB cable, custom recovery .img, custom ROM .zip

Part 1: Unlocking Bootloader

  1. Enable Developer Options: Settings > About Device > Tap HyperOS version 7 times.
  2. Enable OEM Unlocking and USB Debugging.
  3. Bind Mi Account in Mi Unlock Status.
  4. Boot into Fastboot mode and unlock via Mi Unlock Tool.
  5. Reboot (note: data will be wiped).

Part 2: Flashing Custom Recovery via Fastboot

  1. Download the custom recovery image file (e.g., `lineage-recovery.img`).
  2. Boot into Fastboot Mode: Power off, hold Volume Down + Power.
  3. Use the command: 
    fastboot flash boot recovery.img
  4. Reboot into recovery: 
    fastboot reboot recovery

Part 3: Flashing a Custom ROM via ADB Sideload

  1. Download the custom ROM in .zip format.
  2. Boot into recovery mode and perform a factory reset.
  3. ADB Sideload: 
    adb sideload customrom.zip
  4. Wipe Cache/Dalvik (optional), then reboot to system.

Troubleshooting

If stuck in bootloop, perform a factory reset or try re-flashing. Ensure USB Debugging is enabled and your drivers are correctly installed.
