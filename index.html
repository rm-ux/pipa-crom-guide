Full Guide: Unlocking Bootloader, Flashing Recovery & ROM on Xiaomi Pad 6

Prerequisites

  • Xiaomi Pad 6 (HyperOS)
  • A computer with ADB and Fastboot installed
  • Mi Unlock Tool
  • USB cable (good quality)
  • Custom Recovery (.img) file (LineageOS, crDroid)
  • Custom ROM (.zip) you wish to install

Part 1: Unlocking the Bootloader on Xiaomi Pad 6

  1. Enable Developer Options: Go to Settings > About Device, tap HyperOS version 7 times.
  2. Enable OEM Unlocking and USB Debugging in Developer Options.
  3. Link your Mi Account under Mi Unlock Status.
  4. Boot into Fastboot Mode: Power off, hold Volume Down + Power.
  5. Use Mi Unlock Tool to unlock bootloader. Connect device and click Unlock. This process may involve a waiting period.
  6. Reboot the device (all data will be erased).

Part 2: Flashing Custom Recovery via Fastboot

  1. Download the custom recovery (e.g., lineage-recovery.img).
  2. Boot into Fastboot Mode by holding Volume Down + Power.
  3. Use the following command to flash the recovery: 
    fastboot flash boot recovery.img
  4. Reboot into recovery: 
    fastboot reboot recovery

Part 3: Flashing a Custom ROM via ADB Sideload

  1. Download the ROM you wish to install in .zip format.
  2. Boot into recovery (hold Volume Up + Power).
  3. Perform a factory reset: Go to Wipe in recovery, then select Factory Reset.
  4. Enable ADB Sideload: Navigate to Advanced > ADB Sideload and swipe to start.
  5. Sideload the ROM: 
    adb sideload customrom.zip
  6. Wait for the installation to complete. Optionally, wipe cache/dalvik.
  7. Reboot the system.

Troubleshooting and Important Notes

  • If stuck in a boot loop, try factory reset or re-flashing the ROM.
  • If ADB is not recognizing the device, ensure USB Debugging is enabled and the proper drivers are installed.
  • To exit recovery mode, re-flash the stock ROM via Fastboot.

Summary

Unlock the bootloader using Mi Unlock Tool, flash custom recovery using Fastboot, and install a custom ROM via ADB Sideload. Remember, unlocking and flashing will wipe your device, so back up important data.
